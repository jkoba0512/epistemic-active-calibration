# KUKA LBR iiwa 14 R820 — vendored kinematic model

Self-contained copy of the 7-DoF KUKA LBR iiwa arm model used by the iiwa
terminal-controllability case study (`experiments/iiwa_terminal_case_study.py`).

- `model.urdf` — kinematic + visual/collision description (arm links 0–7).
- `meshes/link_{0..7}.{obj,stl,mtl}` — arm link meshes (gripper meshes omitted).

## Provenance

Originally autogenerated by xacro from `lbr_iiwa.urdf.xacro`
(https://github.com/rtkg/lbr_iiwa), redistributed via `pybullet_data`.
